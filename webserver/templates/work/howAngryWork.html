006 : How Angry Twitter Bot

This twitter bot uses the twitter API to search for a given hashtag. After searching that hashtag it outputs 10 tweets and searches each tweet for either postiive or negative words. Using this bot I can take a look at whether or not people are happy or unhappy about a certain topic. Right now, it only searches for a few different words but the fundamentals of the program still stand.

This project taught me about using the twitter API and how to write data collecting scripts. I want to use the data to help teach myself about data analysis and better understand how I can use data to prove an idea
