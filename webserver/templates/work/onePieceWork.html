001 : One Piece Image Classifier

A simple image classfier made using Keras, TensorFlow and openCV. Input an image into the program and it will try its best to guess if it is Luffy, Sanji or Zoro from the anime one Piece. This project used 200 images for each character to train the model. Each image was downloaded using bing image scarper which I also created (and is included in the github repo) using the bing search API. This project taught me about training machine learning models, creating datasets to train my models on as well as how the bing search API works.

This project was heavily inspired by a pokedex, a device in the pokemon games for identifying pokemon.

This project gave me insight into deep learning models. By using a rather small data set I managed to get a fairly accurate program which guesses if a picture is either Luffy, Sanji or Zoro from the anime One Piece. I learnt about creating a dataset and different ways to collect a large variety of images quickly, I created a bing image scarper using the Bing Search API to do so. It also taught me what goes into training a machine learning model. This was made as a first step into the world of machine learning. The two images below show the output of the program, as you can see it guess the image of zoro (right) with 99.9% accuracy, and it guessed the image of luffy (left) with 96.08% accuray
