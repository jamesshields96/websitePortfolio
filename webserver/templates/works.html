05 : Projects

Here is a collection of projects I have undertaken in my spare time. The projects range from a variety of software engineering projects as well as a few black smithing projects. These projects are not finished, however they are in a state where I am happy to show them. I don't think ill ever be able to call them finished, not because I dont have time, but because I want to keep adding to the projects to help me learn.

There is one more project not listed here and that is this website itself. I mainly wanted to learn about back-end development which is the reason I began this, but it slowly became an excellent way for me to present myself and display projects that reflect my skill level. The back-end was created with Python + Flask, while the front end was made with HTML, JavaScript and CSS.

This website is still a work in progress and while it is live I still need to add projects and format the site properly, if you are interested in looking at the projects before I add them to this site, they are on my GitHub account. Currently, the bronze rose and the weather tempreture pages have not been put together, and the twitter bot is only in a draft stage
